# KotlinMvp

描述：一个基于MVP的Kotlin项目，这是比较完整的项目，封装了一些常用工具库<br>
开源库使用：retrofit2+rxJava2+okhttp3+Glide<br>
接口支持：干货集中营-安卓篇<br>
